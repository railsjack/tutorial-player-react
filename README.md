## How to execute
Need to confirm app/package.json  
"main": "./main.dev.js",
```
$ yarn dev
```

## How to build in production mode
Need to confirm app/package.json  
"main": "./main.dev.js",
```
$ yarn build
```

Need to confirm app/package.json  
"main": "./main.prod.js",
```
$ yarn build package-win
```
